# ITO AI 评分系统

ITE 2026 新品系列产品知识考试的 AI 智能评分系统。

## 功能

- **纯前端** — 单页 HTML，无需后端服务器，直接打开浏览器使用
- **38道考题** — 20道单选 + 10道多选 + 8道简答（满分100分，80分及格）
- **AI 智能评分** — 支持 DeepSeek/Gemini/Kimi/ChatAnywhere 四个模型
- **客观题自动判对错** — 单选/多选自动比对标准答案
- **主观题 NLP 评分** — 简答题按采分点 AI 智能评分
- **API Key 本地保存** — Key 存在浏览器 localStorage，安全便捷
- **导出评分报告** — 支持导出评分结果

## 使用

访问 GitHub Pages 链接即可使用。

### 首次使用

1. 点击右上角 **⚙ API Key**
2. 选择 AI 模型并填入对应 API Key
3. 在右侧逐题填写答案
4. 点击 **🤖 AI 智能评分**

### 测试

点击 **📄 样例** 按钮可自动填入标准答案，测试评分功能。
